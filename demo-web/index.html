Interactive Multimodal Sequence-to-sequence

Demo of interactive multimodal neural sequence-to-sequence system based on the toolkit NMT-Keras.

Instructions:

  1. Select a task:
    1. Machine translation.
    2. Image captioning.
    3. Video captioning
  2. Read a source sentence or watch an image or video.
  3. Click on the "Translate!/Transcript!" button for obtaining the automatic translation/transcription for the source sentence, image or video.
  4. If the translation/transcription is not correct, you can make corrections:
    1. Use the mouse or the keyboard arrows to select the position where to introduce the correction.
    2. Type the correction.
    3. Type "/" for introducing an end-of-sentence symbol.
  5. The system will react to these interactions, providing alternative hypotheses.
  6. Once you are satisfied with the translation/transcription, click in the "Validate" button and go to the next sentence.
  7. If Learn from sample is activated (in blue), the system will use this sentence pair for improving the model.

More info

All systems were developed based on the open-source toolkit Interactive-Keras-Captioning.
Interactive pattern recognition is a framework that aims at diminishing the effort required for correcting transcriptions from a system.
For more info, take a look to the following papers:

NMT-Keras: a Very Flexible Toolkit with a Focus on Interactive NMT and Online Learning
Álvaro Peris, Francisco Casacuberta.
In The Prague Bulletin of Mathematical Linguistics vol. 111, pp. 113-124, 2018.

Online Learning for Effort Reduction in Interactive Neural Machine Translation
Álvaro Peris and Francisco Casacuberta.
In arXiv:1802.03594, 2018.

Video description using bidirectional recurrent neural networks
Álvaro Peris, Marc Bolaños, Petia Radeva and Francisco Casacuberta.
In International Conference on Artificial Neural Networks, 3-11, 2016.
